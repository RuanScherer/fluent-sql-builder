# Fluent SQL Builder

## Overview
This is an application that simulate a SQL builder. The purpose of this application is to give me knowledge about builder model and TDD (Test-Driven Development).

## Techonologies
- [Node.js](https://nodejs.org/)
- [JEST](https://jestjs.io/)

## How to use

### Pre-requisites
- [Node.js (v15)](https://nodejs.org/)

## Step-by-step

- Open terminal and run `$ git clone https://github.com/RuanScherer/fluent-sql-builder.git` to clone repository. (requires [GIT](https://git-scm.com/))
- Enter the created folder and run `$ npm install` to install dependencies.
- Run `$ npm test` to run tests.
